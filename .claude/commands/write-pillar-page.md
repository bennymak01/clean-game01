# Write Pillar Page

Write a full service pillar page HTML file for this cleaning business website (麗雅清潔公司).

## Instructions

Match the existing site style (green colour tokens, sticky nav, floating WhatsApp button, branded footer) used in `office-deep-cleaning.html` and `index.html`.

### Content Requirements

- **Keywords**: Target higher search-intent keywords in zh-TW for the service (e.g. 服務名+香港, 服務名+報價, 服務名+推介)
- **Navigation**: Include sticky nav + top bar + breadcrumb linking back to `index.html` and its anchor sections (`#about`, `#services`, `#process`, `#contact`)
- **Photos**: Insert 2 relevant Unsplash stock photos at appropriate positions with captions
- **Structure**:
  - Featured snippet box at the very top (quick answer / 精選摘要)
  - Introduction paragraph
  - H2 / H3 section headings (subtitle hierarchy)
  - Client testimonials written in first-person "I" (我) voice — realistic, specific details
  - At least one bullet / check-list (`✓` style)
  - At least one comparison or data table
  - Real survey or research statistics presented as stat-cards with figures (source cited)
  - At least one expert quote block (金句引用)
  - Conclusion section
  - 2 FAQ items at the end with `schema.org FAQPage` JSON-LD structured data
- **Language**: Traditional Chinese (zh-TW) throughout
- **CTA**: WhatsApp links (`wa.me/85259186741`) in hero, mid-page banner, conclusion, and floating button
- **Footer**: Branded green footer with links back to all `index.html` sections
- **Related links**: Add a "相關服務" section linking back to `index.html#services` and any existing pillar pages

### File Naming

Name the file in lowercase English reflecting the service, e.g. `carpet-cleaning.html`, `disinfection.html`.

### After Creating the File

1. Update `index.html` — make the matching service card clickable (link to new page)
2. Update `index.html` footer — add the new page to the 服務項目 list and 最新文章 list
3. Commit and push to the current branch
